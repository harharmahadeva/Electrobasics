============================================================
BEGINNER-FRIENDLY THEORY | शुरुआती सिद्धांत
============================================================

English
Jumper wires connect points on breadboards and modules. Types: male-male, male-female, female-female. Use color coding: red for positive, black/blue for ground, yellow/green for signal. Good wiring makes debugging easier.

हिन्दी
जम्पर वायर ब्रेडबोर्ड और मॉड्यूल को जोड़ते हैं। रंग कोडिंग से त्रुटियाँ कम होती हैं।

Teacher Tone:
Use short sentences. Avoid unnecessary formulas unless the lesson needs them. Whenever a new technical word appears, Spark should explain it using the glossary.

COMMON BEGINNER MISTAKES | सामान्य शुरुआती गलतियाँ
============================================================

• Random colors
• Loose wires
• Wrong wire type
• Long messy wires

============================================================
TROUBLESHOOTING GUIDE | समस्या समाधान
============================================================

• Push wire fully
• Check row location
• Replace bad jumper
• Use clean routing
