============================================================ REAL LIFE
PROBLEM ============================================================

Imagine spending an hour searching for a single 220Ω resistor.

Professional engineers label, sort and store every component so projects
can be built quickly without mistakes.

============================================================ THEORY
============================================================

Good Lab Organization

• Label every storage box. • Separate resistors by value. • Store
capacitors by type. • Keep ICs in anti-static foam or bags. • Store
sensors and modules in separate compartments. • Use transparent drawer
cabinets. • Return every tool after use.

Never:

❌ Throw mixed components into one box. ❌ Leave PCBs exposed to dust.
❌ Store ICs without ESD protection.

============================================================ PRACTICAL
LAB ============================================================

Tasks

1.  Create labels for your storage boxes.
2.  Separate resistors by value.
3.  Separate LEDs by color.
4.  Place ICs in anti-static storage.
5.  Arrange tools neatly on the workbench.

============================================================ COMMON
MISTAKES ============================================================

❌ Mixing resistor values ❌ Storing ICs loosely ❌ Leaving wires
tangled ❌ No labels on storage boxes

============================================================
TROUBLESHOOTING
============================================================

Can’t find parts?

• Add labels. • Use transparent drawers. • Sort by category first, then
value.

============================================================ QUIZ
============================================================

1.  Where should ICs be stored? Answer: In ESD-safe storage.

2.  Should resistor values be mixed? Answer: No.

3.  Why label storage boxes? Answer: Easy identification.

4.  What keeps the lab efficient? Answer: Organization.

5.  Should tools be returned after use? Answer: Yes.
