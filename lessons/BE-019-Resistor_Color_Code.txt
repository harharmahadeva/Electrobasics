============================================================
LESSON PURPOSE
============================================================

This lesson teaches beginners how to read resistor color bands step by step.
Many resistors do not have numbers printed on them. Instead, they use color bands.
A learner must be able to identify the value before using the resistor in a circuit.

यह पाठ शुरुआती विद्यार्थियों को रेसिस्टर के रंगीन बैंड पढ़ना सिखाता है।
कई रेसिस्टर पर संख्या लिखी नहीं होती। उनकी वैल्यू रंगीन बैंड से पता चलती है।
सर्किट में रेसिस्टर लगाने से पहले उसका मान पहचानना ज़रूरी है।

============================================================
BEGINNER-FRIENDLY THEORY | शुरुआती सिद्धांत
============================================================

English

A resistor color code is a system used to show the resistor's value using colored bands.
Each color represents a number.

A resistor can have:
• 4 color bands
• 5 color bands
• 6 color bands

In this beginner lesson, we focus mainly on 4-band resistors because they are common in starter kits.

------------------------------------------------------------
4-BAND RESISTOR STRUCTURE
------------------------------------------------------------

Band 1 = First digit
Band 2 = Second digit
Band 3 = Multiplier
Band 4 = Tolerance

Example:

Red - Red - Brown - Gold

Step 1:
Red = 2

Step 2:
Red = 2

Step 3:
Brown multiplier = ×10

Step 4:
Gold tolerance = ±5%

So the value is:

22 × 10 = 220Ω ±5%

This means the actual resistor value may be slightly above or below 220Ω.

------------------------------------------------------------
COLOR DIGIT TABLE
------------------------------------------------------------

Black  = 0
Brown  = 1
Red    = 2
Orange = 3
Yellow = 4
Green  = 5
Blue   = 6
Violet = 7
Grey   = 8
White  = 9

------------------------------------------------------------
COMMON MULTIPLIERS
------------------------------------------------------------

Black  = ×1
Brown  = ×10
Red    = ×100
Orange = ×1,000
Yellow = ×10,000
Green  = ×100,000
Blue   = ×1,000,000

Gold   = ×0.1
Silver = ×0.01

------------------------------------------------------------
COMMON TOLERANCE COLORS
------------------------------------------------------------

Brown  = ±1%
Red    = ±2%
Gold   = ±5%
Silver = ±10%

Tolerance tells how much the real resistor value can vary from the marked value.

Example:
A 220Ω resistor with ±5% tolerance can be around:

Lower limit:
220Ω - 5% = 209Ω

Upper limit:
220Ω + 5% = 231Ω

So if your multimeter reads 218Ω or 224Ω, the resistor is still normal.

------------------------------------------------------------
HOW TO FIND THE READING DIRECTION
------------------------------------------------------------

The tolerance band is usually gold, silver, brown or red and is often separated slightly from the other bands.
Hold the resistor so the tolerance band is on the right side.
Then read from left to right.

------------------------------------------------------------
5-BAND RESISTORS
------------------------------------------------------------

A 5-band resistor has:
Band 1 = First digit
Band 2 = Second digit
Band 3 = Third digit
Band 4 = Multiplier
Band 5 = Tolerance

Example:
Brown - Black - Black - Brown - Brown

100 × 10 = 1000Ω = 1kΩ ±1%

5-band resistors are often more precise.
This lesson introduces them lightly. The main beginner focus is 4-band resistors.

STEP-BY-STEP DECODING METHOD
============================================================

Use this method every time:

Step 1:
Find the tolerance band.
Usually gold or silver is tolerance.
Keep it on the right side.

Step 2:
Read the first color.
This is the first digit.

Step 3:
Read the second color.
This is the second digit.

Step 4:
Read the third color.
This is the multiplier.

Step 5:
Read the fourth color.
This is tolerance.

Step 6:
Calculate:
First two digits × multiplier = resistance value.

Step 7:
Verify using a multimeter if available.

Example 1:
Brown - Black - Red - Gold

Brown = 1
Black = 0
Red multiplier = ×100
Gold = ±5%

10 × 100 = 1000Ω = 1kΩ ±5%

Example 2:
Orange - Orange - Brown - Gold

Orange = 3
Orange = 3
Brown multiplier = ×10
Gold = ±5%

33 × 10 = 330Ω ±5%

Example 3:
Yellow - Violet - Red - Gold

Yellow = 4
Violet = 7
Red multiplier = ×100
Gold = ±5%

47 × 100 = 4700Ω = 4.7kΩ ±5%

============================================================
PRACTICAL LAB | प्रैक्टिकल लैब
============================================================

Lab Name:
Decode and Verify Resistors

Required Components:
• 5 through-hole resistors
• Digital multimeter
• Notebook
• Pen or pencil
• Good lighting
• Optional magnifying glass

Safety:
Resistance measurement should be done with circuit power OFF.
Never measure resistance in a powered circuit.

Steps:

1. Pick one resistor.
2. Locate the tolerance band.
3. Place tolerance band on the right side.
4. Read colors from left to right.
5. Decode the value using the color table.
6. Write predicted value in notebook.
7. Set multimeter to resistance mode.
8. Touch probes to resistor leads.
9. Read measured value.
10. Compare predicted value and measured value.

Observation Table:

Resistor No. | Band Colors | Predicted Value | Measured Value | Tolerance OK?
1            |              |                 |                |
2            |              |                 |                |
3            |              |                 |                |
4            |              |                 |                |
5            |              |                 |                |

Expected Learning:
The measured value does not have to be exactly equal to the color code value.
Tolerance explains small differences.

============================================================
COMMON BEGINNER MISTAKES | सामान्य शुरुआती गलतियाँ
============================================================

• Reading the resistor backwards.
• Ignoring the gold or silver tolerance band.
• Confusing red and orange.
• Confusing brown and black in poor lighting.
• Thinking tolerance means the resistor is faulty.
• Forgetting to multiply.
• Confusing 1kΩ with 10kΩ.
• Measuring resistance while the resistor is connected in a powered circuit.
• Touching both resistor leads with fingers while measuring, which can affect high resistance readings.
• Assuming all resistors use 4 bands.

============================================================
TROUBLESHOOTING GUIDE | समस्या समाधान
============================================================

Problem:
The value seems completely wrong.

Check:
• Are you reading from the correct side?
• Did you identify the tolerance band?
• Did you use the correct multiplier?
• Is the resistor still connected to a circuit?
• Is the multimeter in resistance mode?

Problem:
The measured value is slightly different.

Explanation:
That is normal because of tolerance.

Problem:
The colors are hard to identify.

Solution:
Use bright light or magnifier.
Compare with another resistor.
Ask Spark to help if you upload a photo.

Problem:
The multimeter shows OL.

Possible Reasons:
• Resistance is higher than selected range.
• Probe contact is poor.
• Meter is in wrong mode.
• Resistor lead is oxidized or dirty.
