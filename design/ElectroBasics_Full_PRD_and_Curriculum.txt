PROJECT SUMMARY
---------------
Product name: ElectroBasics
Product type: Interactive electronics learning platform
Target users: Beginners, students, hobbyists, diploma/engineering students, makers
Primary platform: Web app / PWA
Future platforms: Android, iOS, desktop wrapper
Core experience: Learn electronics through interactive lessons, real component images,
breadboard activities, simulations, quizzes, projects, and Spark AI guidance.

ElectroBasics should NOT look like:
- A generic admin dashboard
- A school LMS
- Bootstrap/Material dashboard
- A flat CRUD app
- A student project

ElectroBasics SHOULD feel like:
- Brilliant.org + Duolingo + Apple + Linear + an electronics lab bench
- Premium, dark, neon-accented, engineering-inspired
- Visual, practical, interactive, friendly, and confidence-building
