# ElectroBasics Project Blueprint

## Vision

ElectroBasics is an interactive PWA that teaches basic electronics through visual lessons, bilingual explanations, quizzes, practice tests, and Spark, a voice-based AI tutor.

## Mission

Help students understand electronics instead of memorizing definitions.

## Target Audience

- Students
- Freshers
- ITI learners
- Polytechnic learners
- Diploma students
- First-year engineering students
- Hobbyists
- Curious beginners

## Core Principle

Every feature must help the student learn better.

## Spark AI Tutor

Spark is the learning guide inside ElectroBasics.

### Personality

- Calm
- Friendly
- Patient
- Encouraging
- Practical
- Never sarcastic
- Never condescending

### Teaching Style

Spark teaches through:
- analogies
- questions
- hints
- examples
- explanations after mistakes
- English and simple Hindi

### Visual Identity

Spark uses a KITT-inspired red scanning light and voice waveform when speaking.

## Languages

ElectroBasics supports:

- English
- हिन्दी

Hindi should be simple classroom Hindi for North Indian learners. Common electronics terms such as voltage, current, resistance, ohm, ampere, resistor, capacitor, transistor, IC, LED, multimeter, and breadboard can remain in English.

## Security

ElectroBasics follows OWASP Top 10 principles.

- No secrets in GitHub
- No passwords in frontend storage
- Secure backend authentication
- Online progress sync
- Input validation
- Safe rendering

## Development Workflow

- `main` is stable
- `develop` is active development
- feature branches for major work
- every meaningful change must update `CHANGELOG.md`
- every major decision must update `DECISIONS.md`
