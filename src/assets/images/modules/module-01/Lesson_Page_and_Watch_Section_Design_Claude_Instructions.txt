# Example Lesson: BE-001 Welcome to ElectroBasics

============================================================
PURPOSE
============================================================

Build the complete lesson experience for ElectroBasics.

This instruction covers:

1. Lesson Page Design
   - What the learner sees after clicking Start Lesson from Module 01.
   - Desktop design.
   - Mobile design.

2. Section Cards Design
   - The 12 lesson sections.
   - Locked / current / completed states.

3. Watch Section / Start Section Experience
   - What happens when the learner clicks "Watch Section" or "Start Section".
   - Desktop section player design.
   - Mobile section player design.
   - How to place lesson text, images, Spark, quiz and interactions inside the player.

4. Image / asset prompts
   - Real picture prompts.
   - Infographic prompts.
   - UI prompts.
   - Spark mascot prompts.

Important naming decision:
Use button label "Start Section" instead of "Watch Section" where possible.
Reason:
Not every section is a video. Some sections are text, interactive activity, lab, quiz or checklist.
If the UI already says "Watch Section", the behavior should still work the same.

============================================================
GLOBAL DESIGN STYLE
============================================================

Use the same ElectroBasics dark futuristic dashboard design.

Visual style:
- Dark navy / black background
- Cyan, blue, yellow and purple neon accents
- Glassmorphism cards
- Thin glowing borders
- Rounded corners
- Premium electronics-lab feeling
- Subtle circuit-board background pattern
- Smooth hover states
- Large readable text
- Hindi text must be readable
- Beginner-friendly layout

Avoid:
- White page background
- Upgrade / Pro / Premium / Subscription / Pricing buttons
- Crowded text
- Tiny Hindi text
- Too many animations at once
- Random decorative images with no learning purpose

Color rules:
- Background: #050A12 or similar
- Main card background: rgba(8, 18, 32, 0.82)
- Cyan accent: #00E5FF
- Blue active: #1677FF
- Yellow XP/reward: #FFC928
- Green difficulty: #36D66B
- Purple category/badge: #A855F7
- Orange warning: #FF9F1C
- Text primary: #F8FAFC
- Text secondary: #AAB4C0

WATCH SECTION / START SECTION BEHAVIOR

↓
Open Section Player
↓
Load current section content
↓
Show image/visual + English/Hindi text + key points
↓
Spark explains the section
↓
User can Ask Spark
↓
User clicks Mark Section Complete
↓
Current section becomes completed
↓
Next section unlocks
↓
Progress updates
↓
User can open Next Section

Do not hard-code all text into JSX.
Use a data object and render the same SectionPlayer component for each section.
