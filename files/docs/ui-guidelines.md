# ElectroBasics UI Guidelines

## Design Direction

ElectroBasics uses a modern electronics-inspired visual style: dark base, electric blue/cyan accents, rounded glass-like cards, and Spark red motion elements.

## Component Naming

Reusable UI components use the `EB` prefix:

- `EBButton`
- `EBCard`
- `EBInput`

## Accessibility

- Buttons must be at least 44px high.
- Inputs must have labels.
- Focus states must be visible.
- Color contrast must remain readable.
