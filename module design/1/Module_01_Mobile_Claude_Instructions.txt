# ElectroBasics — Claude Instruction File
# Module 01 Detail Page — Mobile Version

============================================================
PURPOSE
============================================================

Build the mobile Module Detail Page for:

Module 01 — Welcome & Electronics Lab
Hindi Title: स्वागत है और इलेक्ट्रॉनिक्स लैब

This screen opens when the learner taps Module 01 from the All Modules mobile screen.

The mobile experience must be compact, readable, beginner-friendly, and optimized for touch.

Important:
Do not include any Upgrade, Pro, Premium, Subscription, Pricing, or Paywall button anywhere.

============================================================
MOBILE DESIGN STYLE
============================================================

Use the same ElectroBasics style:
- Dark navy / black futuristic background
- Cyan, blue, yellow and purple neon accents
- Glassmorphism cards
- Rounded corners
- Thin glowing borders
- Compact stacked layout
- Bottom navigation
- Large tap targets
- Readable Hindi text

Avoid:
- Desktop grid squeezed into mobile
- Tiny text
- Large unreadable posters
- Upgrade / Pro / Premium button
- Too much information in one card
- Horizontal scrolling for core navigation unless it is a carousel

Mood:
Beginner-friendly, safe, guided, premium app experience.

============================================================
MOBILE SCREEN STRUCTURE
============================================================

Use a stacked vertical screen:

1. Mobile Top Bar
2. Module Hero Card
3. Quick Stats Row
4. Progress Card
5. Spark Says Card
6. What You’ll Learn Accordion
7. Lessons in This Module List
8. Unlock Next Module Banner
9. Reward Card
10. Bottom Navigation

Default screen:
Show important summary first.
Lessons list visible without needing deep scrolling.

============================================================
MOBILE TOP BAR
============================================================

Top bar content:

Left:
- Back arrow

Center:
- Module 01

Right:
- Notification icon
- Profile avatar

Back action:
Return to All Modules Overview.

Do not show full desktop search bar here.
Search can be an icon if needed, but not required.

============================================================
BUTTON BEHAVIOR
============================================================

If no lesson completed:
Primary button = Start Module
Action: open BE-001.

If some progress:
Primary button = Continue Module
Action: open next incomplete lesson.

If all complete:
Primary button = Review Module
Action: open BE-001 or module review.

Lesson buttons:
- BE-001 available: Start Lesson
- BE-002 to BE-004 locked initially
- Locked button disabled
- Show requirement text under locked button

============================================================
MOBILE VISUAL RULES
============================================================

Use:
- Dark background
- Cyan neon borders
- Blue active buttons
- Yellow XP/reward
- Green Very Easy difficulty chips
- Purple Orientation chips
- Large readable text
- Spacious touch targets

Avoid:
- Desktop sidebars
- Right sidebar widgets
- Wide tables
- Tiny images
- Unreadable large posters
- Upgrade or Pro cards

============================================================
MOBILE ACCESSIBILITY RULES
============================================================

- Minimum tap target: 44px height.
- Text contrast must be high.
- Hindi text must be readable.
- Locked buttons must include reason.
- Progress must show text, not only visual bar.
- All image thumbnails must have alt text.
- Avoid relying on color alone for status.

- Which module is this?
- How many lessons are inside?
- What can the learner start now?
