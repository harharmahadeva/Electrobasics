============================================================
GLOBAL MOBILE DESIGN STYLE
============================================================

Use the same ElectroBasics dark futuristic app design.

Visual style:
- Dark navy / black background
- Cyan, blue, yellow and purple neon accents
- Glassmorphism cards
- Rounded corners
- Thin glowing borders
- Compact stacked layout
- Bottom navigation
- Large tap targets
- Readable Hindi text
- Beginner-friendly guided flow

Avoid:
- Desktop sidebars squeezed into mobile
- Tiny text
- Large unreadable posters
- Upgrade / Pro / Premium buttons
- Too many details in one card
- Tiny buttons
- Hidden locked reasons

Color rules:
- Background: #050A12
- Main card background: rgba(8, 18, 32, 0.86)
- Cyan accent: #00E5FF
- Blue active: #1677FF
- Yellow XP/reward: #FFC928
- Green difficulty: #36D66B
- Purple category: #A855F7
- Orange warning: #FF9F1C
- Text primary: #F8FAFC
- Text secondary: #AAB4C0

Accessibility:
- Minimum tap target: 44px height
- High text contrast
- Hindi text readable
- Locked buttons explain why locked
- Progress shown with text, not only bar
- Image thumbnails have alt text
- Do not rely on color only for status
