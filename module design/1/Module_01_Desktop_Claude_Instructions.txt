============================================================
PURPOSE
============================================================

Build the desktop Module Detail Page for:

Module 01 — Welcome & Electronics Lab
Hindi Title: स्वागत है और इलेक्ट्रॉनिक्स लैब

This page opens when the learner clicks Module 01 from the All Modules Overview page.

The page must use the same dark futuristic ElectroBasics dashboard design.

Important:
Do not include any Upgrade, Pro, Premium, Subscription, Pricing, or Paywall button anywhere.

============================================================
DESIGN STYLE
============================================================

Use:
- Dark navy / black futuristic background
- Cyan, blue, yellow and purple neon accents
- Glassmorphism cards
- Thin glowing borders
- Rounded corners
- Premium electronics-lab feeling
- Subtle circuit-board background pattern
- Smooth hover states
- Clear readable typography
- Hindi text must be large enough to read

Avoid:
- White page background
- Upgrade / Pro / Premium button
- Crowded text
- Tiny Hindi text
- Too many animations at once
- Random unrelated images

Mood:
Beginner-friendly, safe, welcoming, professional electronics lab.
